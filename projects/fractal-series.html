FRACTAL SERIES

  • YEAR2024
  • ROLE3D GENERALIST

A personal study of fractals generation using Geometry Nodes and Repeat Zone in Blender.

Project Intent

This project explores mathematical fractals as a foundation for procedural 3D design. It translates abstract equations into controllable sculptural systems, producing complex yet visually expressive geometry.

The work is intended to make mathematics feel less intimidating by allowing it to be experienced visually.

Fractal: Iterative Geometry

Fractals emerge from recursive mathematical functions applied over multiple iterations. These functions produce self-similar structures with increasing geometric complexity.

From Equation to Nodes

Fractal equations were researched through documentation and academic references, then reconstructed using Blender’s Geometry Nodes.

Key implementation components include:

  • Iterative point transformations using Repeat Zone.
  • Distance estimation and thresholding.
  • Parameter-driven recursion control.

This node-based approach allows non-destructive iteration and rapid experimentation.

Fractals in Isolation

Bright and saturated materials emphasize fractal's surface complexity and depth.

A dark, subdued background removes distraction, allowing the fractal forms to exist as visual subjects.

The shading approach favors structural clarity and contrast over physical realism.

Reflection & Learnings

Key Learnings

  • Mathematics can function as a strong foundation for generative design.
  • Procedural workflows encourage exploration and iteration.
  • Geometry Nodes bridge theory and visual intuition.
